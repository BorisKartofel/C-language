Авторизационное приложение, которое запрашивает у пользователя логинпароль (заведомо установленный в приложение). 
В случае успешной авторизации выводит приветствие. 
Если пользователь ввел данные три и более раз неправильно, предоставляет возможность ввести пароль опять только через 3 секунд. 
Если пользователь ввел пароль неправильно пять раз - завершиает работу приложения.
